# TypeScript Reinforcement Before React

This project is a small practice workspace built with **Vite** and **TypeScript**.

Its main purpose is to **learn and reinforce TypeScript fundamentals before starting to study the React library**.

## Learning Goal

Before working with React components, hooks, props, and state, this project helps review core TypeScript concepts such as:

- `let` and `const`
- template strings
- object literals
- interfaces
- arrays
- cloning values with the spread operator and `structuredClone`

## Project Structure

The learning examples are located in `src/bases/`:

- `01-const-let.ts`
- `02-template-string.ts`
- `03-object-literal.ts`
- `04-arrays.ts`

The main entry file is `src/main.ts`, where one of the practice files is imported to run in the browser.

## Available Scripts

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## How to Practice

1. Open any file inside `src/bases/`.
2. Modify the example code.
3. Import the file you want to test from `src/main.ts`.
4. Run the project with `npm run dev`.
5. Review the browser output and console logs.

## Next Step

Once these TypeScript basics feel comfortable, the next step is to begin learning **React** with a stronger foundation in typed JavaScript.
